Profile Button Dropdown

Currently the profile button navigates directly to your profile page. How should the dropdown work?

A

Dropdown Menu (replace direct navigation)

Clicking the profile button opens a flyout menu with options:

  • View Profile — navigates to profile page (what the button does now)
  • Settings — opens the new settings page
  • Sign Out — logs out of Spotify

Clean and minimal. Profile is one click away via the menu.

B

Dropdown on chevron, direct click keeps profile nav

Split button: clicking the avatar still goes to profile, but a small chevron/caret next to it opens the dropdown with:

  • Settings
  • Sign Out

Preserves the current one-click profile behavior. Dropdown is secondary.

C

Gear icon next to profile button

Keep the profile button as-is (navigates to profile). Add a separate gear/cog icon button nearby that opens settings directly.

Simplest change. No dropdown needed. But adds another icon to the toolbar.
